# Installation Guide for DescrybeAIPlugin

### Downloading the Extension

Since the DescrybeAIPlugin doesn't have specific releases at the moment, you can download the latest version directly from the repository. Follow these steps:

1. **Navigate to the Repository**: Go to the DescrybeAIPlugin GitHub [repository](https://github.com/JacquelineDMcGraw/DescrybeAIPlugin).

2. **Clone or Download**: 
   - To **clone** the repository, use the following command in your terminal or command prompt:
     ```
     git clone https://github.com/your-username/DescrybeAIPlugin.git
     ```
   - To **download** the repository:
     - Click on the `Code` button on the repository page.
     - Select `Download ZIP`.
     - Once downloaded, extract the ZIP file to your desired location.


2. **Install in Chrome**:
   - Open Google Chrome and go to `chrome://extensions/`.
   - Enable "Developer mode" at the top-right of this page.
   - Click on "Load unpacked" and select the downloaded folder of DescrybeAIPlugin.
   - The extension should now appear in your list of extensions and be available for use.

### Usage

After installing the DescrybeAIPlugin, you can start using it on [Descrybe.ai](https://descrybe.ai) as follows:

1. **Perform a Search on Descrybe.ai**: Simply use the search functionality as you normally would.
2. **Create New or Access Saved Queries**: Click on the extension icon to open the side panel.
3. **Read Away**: Retrieve previously saved selections or add new selections by selecting the checkbox following your desired case.
